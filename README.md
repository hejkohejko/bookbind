Insanely simple script to splice together raw images into printable pages for bookbinding.
Works only on linux <3

# HOW TO USE?

1) Make sure your pages are ordered (possible ordering: alphabetical, by last edit time, +reversals of both)
2) Run BOOKBIND.sh
3) Print generated pages from the 'printable' folder
